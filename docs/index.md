# DotNetHelper-Encryption

This is a simple easy to use hashing & encryption library
that supports the following
- Hashing
    - CRC32
    - SHA1
    - SHA256
    - SHA384
    - SHA512
    - MD5
- SymmetricAlgorithm
    - DES
    - RC2
    - Rijndael
    - TripleDES
    - AES



## Target Frameworks
NET45
NET452
NETSTANDARD
