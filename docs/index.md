# DotNetHelper-Encryption

This is a simple easy to use hashing & encryption library
that supports the following

## Symmetric Support
    DES
    RC2
    Rijndael
    TripleDES
    AES
## Hashing Support
    CRC32
    SHA1
    SHA256
    SHA384
    SHA512
    MD5
## Targeted .NET Frameworks
    NET45
    NET452
    NETSTANDARD
